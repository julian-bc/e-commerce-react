## Installation & Usage
1. Clone the repository:

```bash
	git clone https://github.com/julian-bc/e-commerce-react.git
```

2. Navigate to the project directory:

```bash
	cd e-commerce-react
```

3. Install dependencies:

```bash
	deno install 
```

4. Start the development server:

```bash
	deno run dev
```
